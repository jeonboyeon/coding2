실전 | 카드 유형 디자인 만들기

오늘도 아래 보이는 디자인은 수업을 받으며 피그마로 만들어본 카드유형 디자인 레이이아웃이다.
사이트 제목을 가리고, 박스 안에 3개의 카드를 넣은 디자인이다.

section을 이용해 전체 틀 잡기

제목, 카드 디자인, 카드 안에 요소들이 들어갈 전체 틀을 만들어준다.


html 문서 

<section id="cardType03" class="card__wrap score section">
</section>

css 문서 

/* fonts */
@import url('https://webfontworld.github.io/score/SCoreDream.css');

.score {
    font-family: "SCoreDream";
    font-weight: 300;
}
