javascript의 함수

함수는 자바스크립트의 가장 중요한 핵심이라 봐도 된다.
프로그래밍 언어의 자바스크립트 함수는 어떤 목적을 가진 작업을 수행하는 코드가 모인 블록이다.

함수의 종류

첫번 째 방법 : float : left;
두번 째 방법 : display : flex;
세번 째 방법 : display : grid;

첫번 째 방법 : float

float은 '뜨다'라는 의미가 있는데,
원래는 글과 이미지 요소들을 띄워 배치를 잡기 위해 나온 속성
하지만 예전엔 레이아웃 배치를 잡아줄 속성이 마땅히 없었기에 썼던 속성

레이아웃 틀을 잡을 때 <div>태그로 틀을 잡는데,
<div>태그는 block요소(박스 형식)이기 때문에 각 요소가 세로로 배치를 하게 된다.
이렇게 해서 세로로 배치된 요소들을 위로 끌어당겨 올 때 쓰는 css 속성이 바로 float: left; 속성이다.
이 속성은 부모 요소에 주는 게 아닌 위로 올라와야 할 요소들에만 적용해야 한다.

이 속성을 쓰게 되면 속성을 썼던 아래 요소가 보이지 않는 버그가 생기는데,
해결책으로는 보이지 않는 요소에 clear: both; style을 주게 되면 요소가 다시 보이게 된다.

두번 째 방법 : flex

레이아웃 배치 전용으로 나온 속성 flex
float 보다 작업이 단조롭고 편하다.

flex속성은 float처럼 필요한 요소들에 각각 적용하는 게 아닌,
자식 요소들을 감싸고 있는 부모 요소에게만 적용해주면 된다.
부모 요소에 적용을 시키면 세로로 정렬되어 있던 자식 요소들이 각자 자리에 알아서 들어가게 된다.

부모 요소에 display: flex;을 적용한 후 flex-wrap: wrap; 을 또 적용해주면 레이아웃 배치 끝
flex-wrap: wrap; 속성을 적용하지 않으면 배치가 안 됩니다 :)

세번 째 방법 : grid

grid는 주요 브라우저에서 기본적으로 지원하며, table과 같이 가로 세로를 지정하여 레이아웃을 잡을 수 있는 디자인 주요소이다.

flex와 같이 부모 요소에 display: grid; 속성을 주어 자식 요소들을 컨트롤 하는 방법이다.
display: grid와 세트로 부모 요소에 꼭 들어가야할 속성은
grid-template-areas(요소 이름으로 table과 같이 배치),
grid-template-columns (각 배치 가로 길이 지정), grid-template-rows(각 배치 세로 길이 지정)이다.

자식 요소에는 각각 요소에 이름을 지정해주기 위한 grid-area 속성이 필요하다.

이렇게 레이아웃을 만들 때 사용하는 중요한 속성 3가지 float, flex, grid를 알아봤습니다.
