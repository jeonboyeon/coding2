조건문

조건문은 조건식이 참과 거짓인지에 따라 코드를 제어한다.

if 문

조건이 참일 경우에만 코드를 실행합니다.

if (조건식){
  실행문;
} 

var age = 10;
  var admissionFee = "";
  if (age < 13) {
      admissionFee = "2000원";
  }

  console.log(admissionFee);
결과확인
2000원

if (avg >= 60) {
    if (html < 40) {
