HTML 태그 검색하기

  • a : a 태그는 다른 페이지 이동을 설정한다.
  • abbr : abbr 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • acronym : abbr 머리글자로만 된 단어를 정의할 때 사용 | <abbr>로 바뀌었다.
  • address : 연락할 수 있는 정보를 명시할 때 사용
  • applet : 문서에 포함되는 애플릿을 정의할 때 사용 | <embed> 혹은 <object>로 바뀌었다.
  • area : 이미지 맵에서 하이퍼링크가 연결될 영역을 정의
  • article : 문서나 페이지 또는 사이트와는 완전히 독립적으로 구성할 수 있는 요소를 정의할 때 사용
  • aside : 콘텐츠들로부터 분리시킬 수 있는 콘텐츠로 구성된 페이지 영역을 정의할 때 사용
  • audio : 태그는 음악이나 오디오 스트림과 같은 사운드를 정의할 때 사용
  • b : 굵게 표현되는 텍스트를 정의할 때 사용
  • base : 해당 문서의 모든 상대 주소에 대한 기본 URL과 target 속성값을 정의할 때 사용
  • bdi : 주위의 텍스트들과는 다른 방향으로 써지는 텍스트 영역을 정의할 때 사용
  • bdo : 자식 요소의 텍스트 방향성을 정의할 때 사용
  • bgsound : 배경 사운드 요소 | 현제는 <audio>로 대신 사용 한다.
  • big : 글자 크기가 큰 텍스트를 정의할 때 사용 | 현재는 CSS 로 사용
  • blink : 텍스트를 깜박이게 설정 | CSS Animation으로 대신 사용한다.
  • blockquote : 다른 출처로부터 인용된 블록을 정의할 때 사용
  • body : 문서의 텍스트, 하이퍼링크, 이미지, 리스트 등과 같은 모든 콘텐츠를 포함하는 영역을 정의할 때 사용
  • br : 텍스트 내의 줄바꿈을 정의할 때 사용
